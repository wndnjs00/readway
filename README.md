# 📚 readway
내가 읽은 책을 책장에 저장하고, 책의 페이지수와 리뷰를 쉽게 기록하기위한 목적으로 개발했습니다.
## 개발 기간
2025.11.11 ~ 11.17
<br/><br/>
## ⚒️기술스택
|분류|
|:---:|
`Flutter-dart` `MVVM` `CleanArchitecture` `Riverpod` `Firebase realtime database` `google-books api`


<br/><br/>
